sequenceDiagram
    autonumber
    actor User
    participant Browser
    participant App as ARES Frontend
    participant Server as ARES Backend
    participant DB as Database
    participant Ext as Google/Facebook
    participant LLM as LLM Api

    Note over User, Server: User Chooses Login Method

    alt Option 1: ARES Native Login
        User->>App: Enter Username & Password
        App->>Server: POST /login (creds)
        Server ->> Server: Risk Score Calculation
        opt
          Server ->> LLM: Evaluate this attempt
          LLM ->> Server: Evaluation Resut
          opt Option 1: Found risky
            Server ->> DB: Lock user account
          end
        end
        Server->>DB: Validate Hash
        DB-->>Server: User Valid
        Server-->>App: Return Session Token
    
    else Option 2: Social Login (Google/FB)
        User->>App: Click "Login with Google"
        Note right of App: 1. Constructs Google Auth URL
        App-->>Browser: Redirect (302) to accounts.google.com...
        Note right of Browser: 2. Browser leaves ARES website
        Browser->>Ext: GET /o/oauth2/auth
        Note over User, Ext: User is now seeing the Google Login Screen
        Ext-->>User: Show "Choose Account" Page
        User -->> Ext: Chooses Account
        opt
          Ext -->> User: Asks for additional authentication (2FA)
          User -->> Ext: Validates him/herself
        end
        Ext-->>Server: Callback with Auth Code
        Server->>Server: Exchange Code & Validate
        Server-->>App: Return Session Token
    end
